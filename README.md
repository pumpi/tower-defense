## Features

- Frame-rate independent game loop for consistent gameplay across different devices
- ES6 class-based architecture for clean, maintainable code
- Dynamic enemy waves with increasing difficulty
- Resource management system (coins)
- Canvas-based rendering

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
